# **App Name**: DevFolio

## Core Features:

- Personal Bio & Skills Display: A dedicated section to introduce the user, list their skills, and showcase their experience.
- Interactive Project Showcase: A gallery to display portfolio projects with clickable previews.
- Detailed Project Pages: Individual pages for each project featuring comprehensive descriptions, technologies used, and links to live demos or repositories.
- Integrated Contact Options: Provide clear ways for visitors to contact the user, such as a direct email link or a contact form, and include a prominent contact button.
- Social Media & Professional Network Links: Prominently display links to personal social media profiles and professional networks like Medium, LinkedIn, and GitHub.
- CV Request Functionality: A dedicated button allowing visitors to request or download the user's Curriculum Vitae (CV).
- Responsive Design: Ensure the portfolio is fully responsive and visually appealing across all devices and screen sizes.
- Markdown-Based Content Management: Allow for easy creation and updating of portfolio content (projects, bio) using simple Markdown or MDX files, suitable for GitHub Pages hosting. This includes reading project data from JSON files.

## Style Guidelines:

- Primary color: '#6BA0EE' (a balanced, professional blue) to convey trust and innovation in a modern developer portfolio. Its vibrancy stands out against a dark backdrop.
- Background color: '#1A1C21' (a deep, desaturated blue-gray). This dark scheme provides a modern, high-contrast canvas that is easy on the eyes, aligning with many developer tool themes.
- Accent color: '#31C3E3' (a bright, crisp cyan). This hue is analogous to the primary and offers a striking contrast, ideal for highlighting interactive elements and calls to action without being overwhelming.
- Body and headline font: 'Inter' (a grotesque-style sans-serif) for its clean, objective, and highly readable modern aesthetic, suitable for technical content and general text alike.
- Code font: 'Source Code Pro' (a monospaced sans-serif) for displaying any code snippets with clarity and appropriate developer-centric styling.
- Utilize minimalist, modern line-art icons that subtly enhance content without detracting from the overall clean and professional presentation. Icons should align with a consistent aesthetic.
- Implement a spacious, modular grid-based layout for projects, ensuring excellent readability and visual hierarchy for detailed content sections, maintaining ample whitespace for a clean look.
- Incorporate subtle and smooth transition animations on interactive elements, such as hover effects on project cards and smooth page navigation, to provide a polished user experience.
